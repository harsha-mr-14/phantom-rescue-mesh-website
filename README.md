# 🚨 Phantom Rescue Mesh

### Decentralized Emergency Communication System

Phantom Rescue Mesh is an emergency communication platform designed for disaster situations where internet or mobile networks may fail.

This system allows victims to send **SOS alerts**, **live GPS location**, and **emergency messages** to nearby rescuers using **offline communication technologies like Bluetooth/Wi-Fi Direct**, with optional cloud support.

---

## 🌍 Problem Statement

During natural disasters, remote travel, or network failures:

- Victims cannot call for help
- Mobile networks may fail
- Emergency response becomes slow
- GPS sharing becomes difficult

Phantom Rescue Mesh solves this by creating an **offline rescue communication network**.

## 🚀 How It Works

### Step 1
Victim presses the **SOS button**

### Step 2
Device captures:
- GPS location
- Timestamp
- Emergency message

### Step 3
Alert is transmitted using:
- Bluetooth (offline)
OR
- Firebase (online)

### Step 4
Rescuers receive:
- Alert notification
- Victim message
- Live map location
